# Fault Error Code Comparison Table

| Code | Error Name | Solution |
|:----:|------------|----------|
| E01 | Grounding Fault | Check if the charging station power ground wire is loose or not connected, and reconnect to solve the issue. |
| E02 | Leakage Protection | Check if there is leakage in the charging station power connection, and reconnect to solve the issue. |
| E03 | Input Overvoltage | Voltage is unstable; an excess of 10% above the rated value will trigger an error. |
| E04 | Input Undervoltage | Voltage is unstable; falling below 10% of the rated value will cause an error. |
| E05 | Overcurrent Protection | The current may exceed 16A; use the app to lower the current value. In some European regions, such as Switzerland, where the circuit is only 10A, charging with 16A may cause overheating and damage the circuit. |
| E06 | CP Abnormality | Check if the car-side charging connector is properly inserted, inspect for any foreign objects, and replug if necessary. |
| E07 | Emergency Shutdown | Rotate the reset switch to resolve. |
| E08 | Electronic Lock Abnormality | Charging station with European standard case B (featuring an electronic lock that prompts this message in case of anomalies). |
| E09 | Fire-neutral Line Reversal | Reconnect based on fire-neutral line identification (some sockets, like Schuko sockets, may not differentiate between fire and neutral lines). |
| E10 | Temperature Sensor 1 Fault | — |
| E11 | Temperature Sensor 2 Fault | — |
| E12 | Temperature Sensor 3 Fault | — |
| E13 | Indoor Temperature is Too High | The internal working temperature of the charging station cannot exceed 85°C. |
| E14 | Leakage Protector Verification Error | Unplug and Replug. |
| E20 | Relay Open Circuit | Need after-sales treatment. |
| E21 | Relay Contact Adhesion | After-sales service is required. |
| E22 | Relay Overheating | Need after-sales treatment. |
| E23 | Relay Mechanical Fault | Need after-sales treatment. |
| E30 | EEPROM | There is a problem with the charging pile software. Can it be upgraded via OTA? |
| E31 | MCB Communication Failure | Need after-sales treatment. |
